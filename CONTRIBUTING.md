# Contributing to SiCuci

Terima kasih sudah berkontribusi! 🎉

## 📋 Guidelines

### Code Style
- Gunakan Kotlin coding conventions
- Format code menggunakan Android Studio default
- Buat fungsi/composable dengan nama yang deskriptif

### Commit Messages
Gunakan format:
```
<type>: <description>

type:
- feat: Fitur baru
- fix: Bug fix
- docs: Dokumentasi
- style: Formatting
- refactor: Refactoring
- test: Menambah test
- chore: Maintenance
```

Contoh:
```
feat: tambah fitur backup database
fix: filter minggu ini termasuk 8 hari
```

### Pull Request
1. Buat branch dari `master`
2. Berikan nama yang jelas untuk branch
3. Jelaskan perubahan di PR description
4. Pastikan tidak ada error

## 🐛 Bug Report

Kalau nemu bug, buka issue dengan:
- Judul yang jelas
- Langkah-langkah reproduksi
- Screenshot (kalau ada)
- Device info (model, Android version)

## 💡 Feature Request

Ada ide fitur baru? Buka issue dengan tag `enhancement`!
